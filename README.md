# Student Marks Management System (Java)

A menu-driven Java console application built using arrays and methods.

## Features

- Display student marks
- Calculate total marks
- Calculate average marks
- Find highest marks
- Find lowest marks
- Search marks
- Update marks
- Count pass/fail students
- Count grades

## Concepts Used

- Arrays
- Methods
- Loops
- Switch Case
- Scanner Class
- Menu-Driven Programming
